# Innovexa Catalyst - Freelance Project Marketplace

A full-stack freelance marketplace web application built with React, TypeScript, Tailwind CSS, Express, Prisma, PostgreSQL, JWT authentication, and Socket.io.

## Features

- Client, Freelancer, and Admin role-based authentication
- Project posting and project management
- Freelancer bidding system
- Contracts and milestones
- Escrow payment flow
- Real-time messaging
- Reviews and ratings
- Admin dashboard
- Dark/light mode
- Responsive UI

## Tech Stack

Frontend:
- React
- TypeScript
- Tailwind CSS
- Vite
- Axios
- Socket.io Client

Backend:
- Node.js
- Express.js
- Prisma ORM
- PostgreSQL
- JWT Auth
- Socket.io
